Frontend Developer Leaderboard

Posted 2 hours ago

Worldwide

Summary

We are looking for a frontend developer to build a fullscreen leaderboard screen for a social game bar. Players submit their scores via QR code Google Form Google Sheet. We manually approve scores in the sheet. The leaderboard screen only displays approved scores. This screen will run continuously on a TV inside the bar. Core requirement: rotating leaderboard with animation The screen should automatically rotate between two leaderboards: Crazy Pool Leaderboard Activity Game Leaderboard Rotation behavior: Each leaderboard is shown for 1520 seconds Between leaderboards, a short transition animation is required The animation should be: Subtle Clean Not distracting Suitable for a bar environment Examples of acceptable transitions: Fade in / fade out Slide left / right Quick zoom + fade Short branded interstitial (logo or punchline) No heavy animations, no flashy effects, no gaming UI. Display & UX requirements Fullscreen layout (TV display) Large typography, readable from 68 meters Dark background with neon-style accents Only one leaderboard visible at a time Top 5 entries per game (not more) Auto-refresh data every 3060 seconds Page must be stable for long-running display (hours) Data source Google Sheets (read-only) Only rows where Approved = TRUE are displayed Sorting rules: Crazy Pool: lowest score wins Activity Game: highest score or fastest time wins No backend, no database, no login system. Tech stack HTML / CSS / JavaScript Google Sheets API or published CSV/JSON Clean, maintainable code Deliverables 1 rotating leaderboard webpage Animation/transition between leaderboards Simple configuration for timing (seconds) Instructions on how to connect a Google Sheet Tested in fullscreen browser mode Nice to have (optional) Simple config file for colors/timing Small animation on leaderboard change Ability to easily add a third leaderboard later
